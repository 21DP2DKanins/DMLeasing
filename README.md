# DMLeasing

## Apraksts
DMLeasing ir līzinga vadības sistēma, kas izstrādāta, lai automatizētu līzinga uzņēmuma procesus. Tā ietver funkcionalitāti apskatīt pieejamās automašīnas un iesniegt pieteikumus līzingam.
